# MEDOCODE Portfolio & Templates Platform

A professional developer portfolio and premium templates showcase built with modern web technologies.

---

## Project Overview

MEDOCODE serves as both a high-end personal portfolio and a dedicated Templates Platform. The portfolio showcases professional projects, services, and technical skills, while the integrated Templates Platform allows visitors to browse premium website templates, view their features, and initiate purchase requests directly via WhatsApp. 

The entire platform is designed with a focus on premium aesthetics, featuring full bilingual support (English/Arabic), dynamic theme switching, and fluid responsive design across all devices.

---

## Features

- **Templates Platform**: A dedicated section to browse, search, and discover premium website templates.
- **Template Details**: Comprehensive template pages with feature lists, responsive galleries, and what's included.
- **Fullscreen Gallery Viewer**: Immersive, touch-friendly image galleries with keyboard navigation.
- **WhatsApp Integration**: Seamless one-click "Purchase" and "Customize" flows that auto-generate formatted WhatsApp requests.
- **Multi-language Support**: Fully localized architecture for English (LTR) and Arabic (RTL) reading directions.
- **Dynamic Themes**: Context-driven dynamic toggling between Dark, Light, and Warm premium themes.
- **Responsive Design**: Pixel-perfect layout adaptable across mobile, tablet, and desktop devices.
- **SEO Optimization**: Dynamic meta tags for search engine visibility.

---

## Tech Stack

- **Framework**: React + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **Internationalization**: React i18next
- **Animations**: Framer Motion & GSAP
- **Icons**: Lucide React
- **SEO**: React Helmet Async

---

## Installation

To run this project locally, simply clone the repository and install the dependencies:

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```

---

## Deployment

The project is configured as a static SPA (Single Page Application). To deploy:

1. Run `npm run build` to generate the optimized production files.
2. Upload the contents of the `dist/` folder to your preferred static hosting provider (e.g., Vercel, Netlify, or GitHub Pages).
